## 🎨 UI 컴포넌트 스펙

### 1. 특징 선택 컴포넌트
```typescript
interface FeatureSelector {
  category: string;
  required: boolean;
  options: string[];
  multiSelect?: boolean;
  placeholder: string;
  validation: {
    minSelection?: number;
    maxSelection?: number;
  };
}
```

### 2. 매칭 상태 표시
```typescript
interface MatchingStatus {
  sent: number;       // 보낸 호감
  received?: number;  // 받은 호감 (프리미엄)
  matches: number;    // 성사된 매칭
  displayMode: 'exact' | 'range' | 'hidden';
}
```

### 3. 프로필 공개 단계
```typescript
interface RevealSteps {
  step1: {
    icon: '😊',
    title: '기본 정보',
    reveals: ['nickname', 'gender', 'ageRange']
  };
  step2: {
    icon: '💬',
    title: '관심사',
    reveals: ['bio', 'interests']
  };
  step3: {
    icon: '🔓',
    title: '실제 프로필',
    reveals: ['realProfile', 'photo'],
    requiresConsent: true
  };
}
```

## 🎯 인터랙션 가이드

### 1. 애니메이션
- 호감 전송: 하트 파티클 효과
- 매칭 성공: 컨페티 + 진동
- 특징 매칭: 부드러운 하이라이트

### 2. 피드백
- 실시간 참가자 수 업데이트
- 특징 입력시 예상 매칭 수 표시
- 호감 전송 후 즉각적인 피드백

### 3. 안전 기능
- 익명 신고 버튼 항상 표시
- 차단 기능 즉시 적용
- 모임 나가기 확인 다이얼로그

---

이 UI 디자인을 통해 사용자는 직관적이고 안전하게 즉석 모임 기능을 사용할 수 있습니다.
