# Glimpse 시나리오 테스트 가이드

## 테스트 시나리오 개요

Glimpse 앱의 주요 기능을 검증하기 위한 엔드-투-엔드 시나리오 테스트입니다.

## 테스트 환경 설정

### 1. 사전 준비
```bash
# 데이터베이스 초기화
npm run db:reset

# 테스트 데이터 시드
npm run db:seed:test

# 모든 서비스 시작
npm run dev
```
